How to run:

1. Navigate to /debug/ and open the folder in Powershell

2. Use the following command and syntax to operate:

./UDPClient.exe [-f filename] host port

where:

-f: Optional arg, used if and only if you are sending a .txt file
filename: Used if and only if you are sendig a .txt file, the path of the file
host: The ipv4 of the host
port: The port the host is listening on
